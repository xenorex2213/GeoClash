# 🌍 GeoClash

A fun and interactive geography-based game where players guess locations based on clues. Instead of relying on scoring, the game progressively narrows hints to guide the player toward the correct location.

---

## 🚀 Features

* 📍 Random location generation
* 🗺️ Interactive map for guessing
* 📸 Image-based or street-view clues
* 🔍 Progressive hint narrowing system
* 🎮 Multiple rounds gameplay

## 🎮 How to Play

1. A random location is selected.
2. The player is shown an initial clue (image or minimal map view).
3. The player makes a guess on the map.
4. If incorrect, additional hints are revealed progressively.
5. Hints become more specific over time (e.g., continent → country → region → exact location).
6. The goal is to identify the correct location using the fewest hints possible.

---

## 🔍 Progressive Hint System

The game gradually reveals clues to help the player:

* 🌎 Broad hint (e.g., continent)
* 🌐 Country-level hint
* 🏙️ Region or city hint
* 📍 Precise location clue

This system encourages logical deduction and improves learning of geography over time.
